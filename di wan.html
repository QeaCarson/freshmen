抵玩华师

       再不疯狂我们就老啦！来到大学城只顾学习我们就out啦！Come on up and go！想玩遍整个华师吗？想以最短时间通晓华师内外各大娱乐场所吗？想知道什么样的交通方式才会让你约会不迟到吗？来！U仔今天带你装B带你飞，带你玩遍华师内外！Follow me.

 

 

Part1:华师内

 

1.砚湖书吧
（1）介绍：口味超级多，咖啡吧环境好！里面还有咖啡、果汁，特别适合朋友聚会，同学聚会。
（2）地址：番禺华南师范大学文二栋经济与管理学院210(近教学楼)
（3）营业时间：10:30-22:30

 

2.华师桌球室

  1. 介绍：环境不错，就是一个桌球室。小台收费18元/小时。

  1. 地址：华师生活区北区后勤综合楼(翰园隔壁）3楼
  2. 注意：下午吃晚饭后去的话比较好，因为晚上人比较多，经常会没有位置~
  2. 游泳馆
  3. 介绍：标准型游泳馆，承担过大中型游泳比赛，有两个泳池，一个是训练池，一个是比赛池。外观优美，环境秀丽。冬天还会有暖水呢！
  4. 地址：教学区
  5. 开放时间：每周一三五17:00~18:30
  6. 收费：需凭游泳证进场，每次收费3元。

 

  1. 华师免费健身房
  2. 介绍：华师免费健身房！不用任何凭证就能进去~
  3. 地址：华师体育与科学院楼1楼（网球场对面）
  4. 注意：由于地点离宿舍楼较远，很多同学都觉得走到过去已经满头大汗了，不过这样的资源真的很难得！
  2. 鸿星健身俱乐部
  3. 介绍：是一个比较大众化的健身房，对于学生来说还不错。价格很便宜，年卡只要几百块。平时晚上会有一些课程可以上，但是器材不是特别多，损坏程度也比较严重，跑步机有6-8台。
  4. 地址：华师生活北区后勤综合楼翰圆三楼(近星海宿舍后门，佳来福超市旁)

 

  2. California吧＆果间艺术吧
  3. 介绍：两间清吧。California吧装修较新，场地较大，有几台大电视和一个大舞台，最适合社团搞活动，平时一群朋友在这里聚会也是个不错的选择；果间艺术吧相对较小，但比较有情调，食物也很好吃，是个拍拖的好地方。
  4. 地址：华师生活北区后勤综合楼二楼（果间艺术吧）和三楼（California吧）


Part2:华师外


1.中心湖公园
（1）介绍：中心湖位于大学城小谷围岛中心，可谓是大学城的心脏，以其为中心辐射向外，带有环湖路，内环路等，可以见到大学城各高校的景色，夜晚的时候很多大学生在此聚会。平时也有很多人在旁边散步、遛狗、放风筝······
（2）交通：从内环西路车站（华师南6栋宿舍楼下）往省中医院方向步行500米左右即可。
（3）推荐：由于公园里有个广阔的草坪，所以在风大的日子非常适合放风筝，风筝在公园里就能买到。同学们都说，在大学城的情侣都一定要去一下中心湖呢！

2.GOGO新天地商业广场：
（1）吃喝：一楼至二楼都是购物区，更有麦当劳、七十一便利店、万宁、绿叶居、皇冠等，是广大女性、情侣约会的好去处！三楼是美食广场，几十家快餐厅，几十种地方风味，保证能满足嘴刁的你！
（2）玩乐：四楼是V-BOX 带自助餐的KTV，闺蜜、社团各种聚会都可以在这里痛快唱一宵！
PS：整家商场的消费水平都很适合大学生哦！如果你是土豪，这里绝对可以一次性满足你N个要求~

另外，每年的跨年盛会都能吸引成千上万的学生哦！

 

（3）交通方式（乘车默认以星海音乐学院站为起点，中环东路为终点）：

  1. 从星海学院站(大学城)乘坐 382路 (坐3站)、 番201路 (坐3站)到中环东路站(大学城)下。走约250米到GOGO新天地商业广场。
  2. 从星海学院站(大学城)乘坐 大学城4线 (坐2站)、 B25路 (坐1站)、 801路 (坐2站)、 大学城2线 (坐1站)到地铁大学城北站(公交站)下。走到大学城中环东路。注意,请留意以下线路详情：大学城2线(约30分钟一班车，07:00-22:00)
  3. 从星海学院站(大学城)往前走约680米到综合商业北区站(大学城)乘坐 361路 (坐2站)到中环东路站(大学城)下。走约250米到GOGO新天地商业广场。
  4. 步行：从南区的篮球场往北区方向沿直线一直走。

 

3.北亭广场

  1. 吃喝：一楼有麦当劳、寿司。
  2. 玩乐：3楼新派KTV（上去的时候留意一下楼下有没有代金券可以拿，能省很多哦！也可以问人借会员卡，但每张卡只能开一个房）
  3. 地址：大学城外环西路368号。从华师西门（靠图书馆那个）出门往右走约500米即可。

 

4.广大商业中心

  1. 吃喝：有必胜客、KFC，也有稍高级的车库，里面一楼也有很多快餐店、大排档，简直就是广大学生的第五个饭堂！

（2）玩乐：ABC量贩式KTV

  1. 介绍：总体一般，但好在便宜，团购更便宜！自助餐很一般，不推荐。
  2. 地址：广大商业中心C区3楼(近中环路)
  3. 注意：因为人多，最好提前打电话订房。电话：39341118
  4. 交通：几乎什么车都能去到广大那边，如果不嫌弃还可以走路过去~

 

5.小洲村
（1）吃喝：

  1. 可以到金瀛酒家喝早茶！里面无论是环境还是点心都十分传统！每天早上都会有很多老人家在那里，当然也有很多大学生一早就骑车过去。价格十分实惠哦!注意由于是早茶，所以建议8点左右最好就去到啦~

  1. 有很多烧烤场，会提供各种烧烤需要用的东西，有的甚至可以提供麻将！ 村里也有很多特色小吃的摆摊，有砵仔糕、麦芽糖等！
  2. 玩乐：小洲村里面有不少小资小清新的景点和店铺，就像是山寨的鼓浪屿一般！没有去过鼓浪屿的可以先到小洲村预热一下！

 

  1. 交通：华师学生推荐在学校里面组自行车骑过去，具体路线可查询各种网上地图。
  2. 岭南印象园
  3. 介绍：是集观光、休闲、娱乐、住宿、餐饮、购物，体验岭南乡土风情和岭南民俗文化的旅游景区，富有特色的街巷、宗祠、民居和店铺等，充分展现了岭南传统文化的精华。一天到晚有很多表演看，就能赚回票价了!
  4. 地址：在大学城南附近，算是一个比较偏僻的地区了吧，下了地铁之后要走一段路。票价：成人票为60元/人，学生票40元/人，亲子票90元/两人，有时候会有团购（网上预订学生票35元/人）

 

  1. 科学中心
  2. 介绍：省级的科普教育基地，星期六日很多小孩子过来参观。实际参观主要是分为两层，第一层是少儿的科普展览，第二层收费，主要是世博广东馆、广东历史馆、LED的使用推广等几个主题，还可以观看科普电影！场馆很大，建筑也气派。
  3. 交通：从星海学院站乘坐 383路 (坐5站)、 801路 (坐5站)到大学城总站(科学中心)下。走约380米到广东科学中心(大学城)北门。
